# Clinicaped App - Mobile

Aplicativo mobile desenvolvido com React Native para cadastro de usuários, listagem de clínicas e sistema de agendamento de consultas.  

Este app é parte do sistema completo Clinicaped, junto com a API feita em NestJS.

## Tecnologias Utilizadas

- React Native (Expo)
- React Navigation
- React Native Paper
- Axios
- Node.js (API externa)
- JWT para autenticação (via API)

## Funcionalidades

- Login e cadastro de usuários
- Listagem de clínicas disponíveis
- Marcação de consultas
- Interface adaptada para Android e iOS
- Tela administrativa ao logar como `adm@gmail.com`
